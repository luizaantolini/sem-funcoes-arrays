# Exercícios sem funções de arrays

Este repositório contém exercícios em JavaScript que devem ser implementados sem utilizar as funções prontas de manipulação de arrays. Os exercícios foram desenvolvidos para praticar estruturas de repetição, condicionais e manipulação manual de vetores.

**Tarefas** 📝

1) Leia 7 nomes de pessoas e guarde-os em um vetor. No final, mostre uma listagem com todos os nomes informados, na ordem inversa daquela em que foram informados. 7️⃣

2) Leia um vetor com 8 números reais e calcule a média dos valores armazenados. Em seguida, exiba os valores do vetor que estão acima da média. 🔢

3) Leia o nome e a idade de 9 pessoas e guarde esses valores em dois vetores, em posições relacionadas. No final, mostre uma listagem contendo apenas os dados das pessoas menores de idade (nome e idade).🛰️

4) Leia dois vetores de 5 elementos cada (valores inteiros). Crie um terceiro vetor que contenha apenas os elementos que estão em ambos os vetores (interseção, sem repetição). 🤖
